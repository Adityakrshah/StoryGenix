## âš™ï¸ Installation & Setup

### Clone the repository

```bash
git clone https://github.com/Adityakrshah/StoryGenix.git
cd StoryGenix
```

### Backend setup

```bash
cd backend
npm install
```

Create a `.env` file:

```env
PORT=5000
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_jwt_secret
GROQ_API_KEY=your_groq_api_key
```

Run backend:

```bash
npm start
```

### Frontend setup

```bash
cd frontend
npm install
npm run dev
```
